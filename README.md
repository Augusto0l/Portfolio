# Portfólio de Projetos

Este repositório reúne projetos desenvolvidos durante minha trajetória de aprendizado em programação, banco de dados, automação e desenvolvimento de sistemas. A proposta é organizar, documentar e apresentar projetos simples, intermediários e mais complexos, demonstrando minha evolução prática na área de tecnologia.

Os projetos presentes neste portfólio foram criados com o objetivo de aplicar conhecimentos adquiridos em cursos, estudos independentes, atividades acadêmicas e práticas pessoais. Cada projeto representa uma etapa do meu desenvolvimento, explorando conceitos como lógica de programação, automação de tarefas, manipulação de dados, desenvolvimento web, integração com bibliotecas externas e construção de soluções computacionais.

## Objetivo

O objetivo deste portfólio é centralizar meus projetos em um único repositório, facilitando a visualização da minha evolução como estudante de Ciência da Computação e servindo como forma de exposição profissional no GitHub.

## O que você encontrará aqui

Neste repositório, é possível encontrar projetos envolvendo:

- Python;
- Automação de tarefas;
- Lógica de programação;
- Manipulação de arquivos e dados;
- Desenvolvimento web;
- Web scraping;
- Pequenos sistemas e scripts práticos;
- Projetos acadêmicos e pessoais.

## Organização

Cada projeto possui seus próprios arquivos e, quando necessário, uma documentação explicando seu funcionamento, objetivo, tecnologias utilizadas e forma de execução.

## Sobre mim

Sou estudante de Ciência da Computação e tenho interesse em desenvolvimento de software, banco de dados e segurança digital. Utilizo este portfólio como forma de registrar minha evolução, praticar novas tecnologias e apresentar projetos que demonstram minhas habilidades técnicas.

## Status do Portfólio

Este portfólio está em constante atualização. Novos projetos serão adicionados conforme avanço nos estudos e desenvolvo novas aplicações.
